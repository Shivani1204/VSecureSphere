Welcome to Experiment 6: Intrusion Detection with Snort IDS

Introduction to Snort IDS

Snort is a powerful open-source Intrusion Detection System (IDS) developed by Martin Roesch and now maintained by Cisco. It’s one of the most popular tools used by cybersecurity professionals to detect and prevent network intrusions.

Snort operates by analyzing network traffic in real-time and comparing it against a predefined set of rules. When it detects suspicious behavior—like port scanning, DoS attempts, or brute-force attacks—it logs alerts to help identify and respond to threats.

Core Features of Snort:

  • Real-time packet analysis and logging
  • Protocol decoding and content matching
  • Customizable rule-based detection
  • Detection of common threats, such as port scans, buffer overflows, and brute-force attempts

In this experiment, you’ll deploy Snort on a victim Ubuntu container and simulate various attacks using a Kali Linux attacker container. You’ll then observe how Snort detects and reports these attacks in real-time.

This lab provides hands-on experience in working with Snort IDS and helps build a deeper understanding of network security monitoring and threat detection.
