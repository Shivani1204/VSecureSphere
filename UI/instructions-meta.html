Meta-Exploit Vulnerability Scan Instructions

Follow the steps below to perform a vulnerability scan using Metasploit on a specific target, such as the victim system.

Step 1: Login to Both Systems

  • Attacker System: Password: VSecureSphere
  • Victim System: Password: VSecureSphere

Step 2: Verify Victim is Running

  • Login to the victim system.
  • Open a terminal and run the following command to verify the IP address: 
    ifconfig
  • Ensure that services like Apache (port 80) and OpenSSH (port 22) are running: 
    service apache2 start
    service ssh start
    service vsftpd start

Step 3: Launch Metasploit on the Attacker

  • Open a terminal on the attacker system.
  • Run the command to start Metasploit: 
    msfconsole

Step 4: Perform Recon on the Victim

  • Scan for open FTP version: 
    use auxiliary/scanner/ftp/ftp_version
    set RHOSTS   # Replace with the actual victim IP
    run
  • Check if anonymous FTP login is allowed: 
    use auxiliary/scanner/ftp/ftp_anonymous
    set RHOSTS   # Replace with the actual victim IP
    run

Step 5: Perform TCP Port Scan

  • Run a TCP port scan on the victim: 
    use auxiliary/scanner/portscan/tcp
    set RHOSTS   # Replace with the actual victim IP
    set THREADS 50
    set PORTS 1-1000
    run

Step 6: Perform SYN Scan

  • Perform a SYN scan for stealthier port detection: 
    use auxiliary/scanner/portscan/syn
    set RHOSTS   # Replace with the actual victim IP
    set THREADS 100
    set PORTS 1-1000
    run

Step 8: Final Tips

  • Validate open ports and services.
  • Use Metasploit payloads to maintain access (e.g., reverse shell).
  • Adjust thread count for faster scanning in larger networks.
