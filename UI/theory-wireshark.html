Theory

Purpose of the Experiment:

The objective of this experiment is to understand how to use Wireshark to capture, analyze, and interpret network traffic for troubleshooting and security assessment. By examining network packets, users can gain insights into how data flows across a network, identify potential issues, and enhance security measures.

What is Wireshark?

Wireshark is a free, open-source network protocol analyzer that allows users to inspect data packets in real time. It is widely used for network monitoring, performance analysis, troubleshooting, and cybersecurity investigations. By capturing network traffic, Wireshark helps in:

  • Monitoring network activity to understand data flow and communication.
  • Diagnosing network issues such as high latency, dropped packets, and misconfigurations.
  • Detecting security threats like unauthorized access, malware communication, and data breaches.
  • Analyzing protocol behavior to understand how different network services interact.

Key Concepts:

1. Packet:

A packet is a small unit of data transmitted over a network. Each packet contains essential components, including:

  • Source and Destination Addresses: Identify the sender and receiver of the packet.
  • Data Payload: The actual information being transmitted.
  • Protocol Information: Defines how the packet should be handled (e.g., TCP, UDP).
  • Headers and Footers: Contain metadata for routing and error checking.

2. Protocol:

A protocol is a set of rules that govern how data is transmitted and received over a network. Common network protocols include:

  • TCP (Transmission Control Protocol): Ensures reliable, ordered, and error-checked data transmission.
  • UDP (User Datagram Protocol): A faster but less reliable communication method, commonly used for streaming.
  • HTTP/HTTPS (Hypertext Transfer Protocol): Used for web browsing and data transfer.
  • ICMP (Internet Control Message Protocol): Used for diagnostic purposes (e.g., ping requests).

Wireshark enables users to capture and analyze network packets, helping them understand communication patterns, identify network vulnerabilities, and improve overall network security.
