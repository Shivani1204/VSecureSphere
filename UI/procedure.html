Procedure

Step 1: Install Nmap

To install Nmap, update your package list and install Nmap using the following commands:

            sudo apt-get update
            sudo apt-get install nmap

Type the above commands in the terminal.

Step 2: Find Your Network IP Range

Use the following command to find your IP address and network range:

            ifconfig

Look for an entry like 192.168.1.xxx to determine your IP address.

Step 3: Run a Basic Network Scan

Now that you know your network range, scan for active devices using Nmap:

            nmap 192.168.1.x/xx

This command tells Nmap to scan all devices connected to your network.

Step 4: Identify Open Ports

After scanning, Nmap will provide a list of detected devices. Each device will display:

  • IP Address: The unique identifier for the device.
  • Host Name: The name of the device, if available.
  • Open Ports: The network ports that are open on the device.

Step 5: Find Open Ports on a Specific Device

To check which ports are open on a specific device (e.g., 192.168.1.10):

            nmap -sV 192.168.1.10

The -sV option tells Nmap to detect the services running on open ports.

Step 6: Example Output

After executing the above commands, you will see output similar to the following:

            PORT     STATE  SERVICE
            22/tcp   open   ssh
            80/tcp   open   http
            443/tcp  open   https

Explanation:

  • 22/tcp (SSH): Used for remote login.
  • 80/tcp (HTTP): Used for web browsing.
  • 443/tcp (HTTPS): Used for secure web browsing.
