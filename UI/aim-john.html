Aim

To perform password cracking and analyze password security using John the Ripper.


Prerequisites

  • Basic knowledge of password hashing algorithms and how they work.
  • Familiarity with Kali Linux and its command-line interface.
  • Understanding of how brute-force and dictionary-based attacks operate.
  • Both Attacker and Victim machines must be set up and running with access to the hashes.
  • Basic understanding of John the Ripper and its configuration files.
  • Ensure that the necessary wordlists (e.g., rockyou.txt) are available for password cracking.
