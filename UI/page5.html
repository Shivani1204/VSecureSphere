Welcome to Experiment 5: Phishing Attack Demonstration

Introduction to Phishing

Phishing is a type of social engineering attack often used by cybercriminals to trick individuals into providing sensitive information such as usernames, passwords, credit card details, or installing malicious software. The attacker usually masquerades as a trustworthy entity via fake websites, emails, or messages to lure the victim.

In this experiment, we demonstrate a phishing attack using a Kali Linux Attacker Machine and an Ubuntu Victim Machine. The attacker hosts a fake login page and uses SET (Social Engineering Toolkit) to clone a legitimate website. When the victim enters their credentials, they are captured and displayed on the attacker machine.

Objective:

  • To understand how phishing websites work
  • To demonstrate credential harvesting using SET
  • To simulate an attacker-victim scenario in a controlled environment

Tools Used:

  • Kali Linux (Attacker Environment)
  • Social Engineering Toolkit (SET)
  • Ubuntu (Victim Environment)
  • Docker & Minikube (Containerized Deployment)
  • noVNC (Web-based VNC Interface)

This experiment is intended for educational purposes only. The goal is to raise awareness about phishing techniques and how they can be prevented through user awareness and security best practices.
